# Atlas Commerce Website

A clean, professional website for Atlas Commerce with pages for hosting, web design, e-commerce, and contact.

## Files
- `index.html` — Homepage
- `hosting.html` — Hosting plans
- `design.html` — Web design packages
- `ecommerce.html` — E-commerce packages
- `contact.html` — Contact and lead form
- `assets/css/styles.css` — Site styling and responsive layout
- `assets/js/main.js` — Mobile menu toggle

## Previewing the site
1. Open the workspace in VS Code.
2. Open `index.html`.
3. Use the Live Server extension or right-click and choose `Open with Live Server` to preview in your browser.
4. Alternatively, open the file directly in a browser.

## Customizing images and logo
- The site is already using a logo asset at `assets/images/logo.svg`.
- Replace `assets/images/logo.svg` with your actual logo file, or update the header HTML if you want to use another image format.
- Add your own pictures by editing the page HTML and referencing images from the `assets/images/` folder.
